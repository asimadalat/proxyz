# PROXYZ

PROXYZ is a systems programming language focused on expressive syntax and memory safety for OS-level development.

## Project Goals
- **Simplicity:** High-level ergonomics, no hidden performance costs.
- **Exploration:** Investigating ownership models for the AXIS OS kernel.
- **Bootstrapping:** Long-term roadmap involves transition from the current Rust implementation (`pxzc-rust`) to a self-hosted compiler.
